C++ Compilers

Okay, now that you have CMake and Visual Studio Code, you will need to get a C++ Compiler. There are many options and various ways to obtain compilers. For Windows, a good C++ compiler can be obtained by downloading Visual Studio, and NO, its not VS-Code. Visual Studio has a free version called community edition, so download that one if needed. While installing Visual Studio, make sure you select the C++ Desktop Development Workload. You can always change this later if needed. You simply can run the installer again and it will let you modify and add more workloads. For Linux, most of the distros already come with compilers, and for Mac, you may also already have one, either way they options are listed below.

This is not a full list of compilers, feel free to look around, however, these are popular choices and among the best.

Windows

Compiler Description
MSVC (Microsoft Visual C++) - Download the Community Edition and select C++ Workload within the installer
MinGW (Minimalist GNU for Windows) - Ports GCC to Windows
Clang Available via LLVM, often used with MSVC or MinGW

Linux

Compiler Description
GCC The most widely used compiler on Linux
Clang LLVM's compiler, an alternative to GCC with better error messages and modularity

Mac

Compiler Description
Xcode Clang Apple's default compiler, included with Xcode and Command Line Tools
Clang LLVM's Clang, available via Homebrew or direct installation
GCC Available via Homebrew, but often just an alias for Clang
