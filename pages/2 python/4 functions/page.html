Builtin Functions

  • print() - Outputs text or variables to the console
  • input() - Reads user input from the console
  • len() - Returns the length of an iterable (e.g., list, string)
  • type() - Returns the type of an object
  • int(), float(), str() - Converts values to integers, floats, or strings
  • round() - Rounds a number to a given precision
  • abs() - Returns the absolute value of a number
  • max() / min() - Returns the maximum/minimum value from an iterable
  • sum() - Returns the sum of a sequence of numbers
  • all() - Returns True if all elements in an iterable are truthy
  • range() - Generates a sequence of numbers
  • enumerate() - Returns an iterable with index-value pairs
  • zip() - Combines multiple iterables element-wise
  • map() - Applies a function to each item in an iterable
  • filter() - Filters elements of an iterable based on a condition
  • sorted() - Returns a sorted list from an iterable
  • reversed() - Returns a reversed iterator
  • isinstance() - Checks if an object is an instance of a specific type
  • id() - Returns the memory address of an object
  • hash() - Returns a hash value for hashable objects
  • open() - Opens a file for reading/writing
  • eval() - Evaluates a string as Python code (use with caution)
  • exec() - Executes a block of code
  • dir() - Lists attributes and methods of an object
  • help() - Displays documentation for an object
  • globals() / locals() - Returns global/local symbol tables as a dictionary
  • hasattr() - Checks if an object has a given attribute
  • setattr() - Sets an attribute on an object
