Git

Here are some common git commands.

  • git init - Initialize a new Git repository.
  • git clone - Clone a repository from a remote source.
  • git add - Stage changes for commit.
  • git commit -m "" - Commit staged changes with a message.
  • git status - Show the status of the working directory.
  • git log - View commit history.
  • git branch - List branches or create a new one.
  • git checkout - Switch to another branch.
  • git switch - Alternative to checkout for switching branches.
  • git merge - Merge a branch into the current one.
  • git pull - Fetch and merge changes from a remote repository.
  • git push - Push local commits to a remote repository.
  • git fetch - Retrieve updates from a remote repository without merging.
  • git rebase - Apply commits on top of another branch.
  • git reset - Unstage changes or reset commits.
  • git stash - Temporarily save uncommitted changes.
  • git tag - Create a tag for a specific commit.

GitHub

GitHub is an online service that allows you to host your repositories through varrious request methods.

GitHub Desktop

GitHub Desktop provides a user-friendly graphical interface (GUI) that allows you to perform common Git operations—such as cloning repositories, committing changes, managing branches, and syncing with remote repositories—without needing to memorize or type Git commands. This can be especially helpful for those who are new to Git or prefer a more visual approach to managing their projects.
