Editors

Visual Studio Code (VS Code) is one of the easiest and most popular choices for Python development today. It offers many free extensions that enhance Python support within the editor. Another option (but more as a side companion) is Jupyter Notebooks, a web-based interface for Python that allows running code in isolated blocks rather than executing an entire script. VS Code also supports Jupyter Notebooks through an extension, enabling the same functionality within the editor. Jupyter Notebooks are particularly useful for fast iteration and interactive coding. Watching some tutorial videos can help give you an idea for the differences but essentially you might prefer using both.

Windows Help

When installing python, make sure to enable "Add Python to PATH" option. This ensures the system paths will include python.

You can check if your system recognizes python as a command. Just open Command Prompt and type "python". It should start the python interactive interpreter in the command prompt if working. You can exit the python interpreter by typing quit(), returning you back to the command prompt. You can also type "echo %PATH%" into command prompt to see if python is in the system paths.

On windows, you can open the System Properties and edit the Enviroment Variables manually if needed. Just make sure to close command prompt or any terminals for the updates to take effect. NOTE You want to add the python directory and the python/Scripts directory. The Scripts sub directory includes the python PIP script which allows you to easily use pip install commands to install 3rd party modules.
